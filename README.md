# Insight2Action

Agentic AI system to turn Google reviews into actionable insights for restaurants and product owners.

## Project Structure
- **main_workflow.py**: Entry point for orchestration
- **config/**: Configuration management
- **adk_agent/**: Agent definition and tools
- **data_processing/**: Data writing utilities
- **reporting/**: Notification and reporting services

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure `.env` and `config/settings.py`
3. Run: `python main_workflow.py`
